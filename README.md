# 📊 Vendor Invoice Intelligence Portal

An AI-powered web application built using Streamlit to automate freight cost estimation and detect risky vendor invoices.

---

## 🚀 Features

- 📦 Freight Cost Prediction  
  Estimate freight cost based on invoice value.

- ⚠️ Invoice Risk Detection  
  Identify invoices that may require manual approval using a machine learning model.

---

## 🧠 Technologies Used

- Python  
- Streamlit  
- Pandas  
- Scikit-learn  
- Joblib  

---

## 🎯 Objective

The main objective of this project is to automate freight cost estimation and detect risky vendor invoices, helping businesses reduce manual effort and improve financial decision-making.

---

## 📊 How It Works

1. User enters invoice details  
2. System processes the data  
3. Outputs:
   - Estimated Freight Cost  
   - Invoice Risk Status (Safe / Manual Approval)  

---

## ⚠️ Note

- Freight prediction currently uses a basic logic (5% of invoice value)  
- Invoice risk detection is powered by a trained machine learning model
